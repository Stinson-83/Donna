Paper, ink, and one moment of rust.

Six core colors. One warm paper background, two warmer surfaces, ink for all text, a single rust accent, and two neutral muted levels. Signals (success, warning, danger) are pulled from the same editorial family — no new hues introduced.

04 · How to use it

The rules.

One rust moment per screen

A screen has exactly one rust · one word in a heading, or one accent button, or one active chip. Not two. If two things compete for rust, the screen has two priorities and the design is wrong.

Ink does the hierarchy

90% of type is ink #1E1A18. Hierarchy is earned by size and weight, not by color. Muted #6B615C is the only lighter text allowed, and only for secondary.

Paper, surface, hairline — the three card treatments

Surface (filled #F0EAE4) = primary card. Hairline (paper bg + 1px ink-200 border) = secondary card. A page holds at most two of each before it reads as busy.

No gradients, no shadows by default

Elevation is a hairline or a surface swap · never a drop shadow at rest. Shadows appear only on modals and floating toasts.

Never introduce a new hue

If a new color feels needed, the answer is a scale step of paper, ink, or rust · or a signal. The palette does not grow.
