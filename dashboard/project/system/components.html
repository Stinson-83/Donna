02 · Components

Every part she’s made of.

Specified, stated, and assembled. Each component uses only the tokens on the foundations page · padding, color, radius, type · so the family resemblance holds everywhere.

Buttons

Four variants. One primary action per surface. The rust button is reserved for accept / promise / keep moments · the one thing that matters most.

Primary · ink

The default for 90% of actions. bg: --ink, color: --paper-100, radius: --radius-sm, padding 12px 20px.

Accent · rust

Reserved for the single keeping action on a screen · saving a memory, confirming she should remember, accepting her suggestion. One per screen, always.

Secondary

Paper background, hairline border. Pairs with primary to offer a quiet decline. Never two secondaries next to each other.

Ghost · text link

No chrome. Sans, medium weight, with a small arrow that translates on hover. For tertiary actions and “see more”.

Inputs & the composer

The composer is the most-used surface in the product. It should feel like an open page, not a form field. Placeholder text is always serif italic · she’s suggesting, not labeling.

She’ll only use it when it matters.

The lockup

One wordmark. One mark. One dot. donna is always set in EB Garamond italic, lowercase, with a single rust period at the baseline · the small thing she keeps after everything else. The mark is that same idea compressed: a lowercase italic d in a hairline circle, with the dot just off the edge. Avatar in conversation. Watermark on assets.

Clear space

Minimum clear space around the wordmark = the x-height of the lowercase letters. Nothing enters that halo.

The rust dot

Not a period. It’s her · the small thing she keeps after everything else. Baseline, offset by 0.12em. Scales with the wordmark, never alone, never recolored.

Case

Always lowercase. Never DONNA, never Donna. The softness is the point.

Weight & style

EB Garamond · italic · regular (400). Never bold, never roman, never another serif. If italic is unavailable, nothing is.

List rows

Divided by hairlines, never boxed. Comfortable density · space-4 vertical padding. One primary line, one sub, one right-aligned action or status.
