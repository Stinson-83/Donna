02 · The two families

One serif. One sans. Each with one job.

EB Garamond · for hierarchy

Upright, weight 400. Used for H1, H2, Display. The serif holding the reader's eye on a page title or a section opener. Not used for emotion, mood, or decoration.

Weight 500 is allowed for H3, where the type sits smaller and needs a touch more stem weight to hold its own at 22px.

Serif is never bold (700). If you need more emphasis than 500, the layout is wrong.

Red Hat Text · for the product

400 for reading, 500 for action, 600 for card titles. Body copy, buttons, inputs, labels, metadata, chat, navigation · everything the user reads in order to do something.

Sans is never italic in UI. Italic sans reads as apology or error state and Donna is neither.

Italic · the single accent

Italic is a highlight, not a default. It appears in exactly three places:

· Inside running body copy, for a book title or a proper name of something.
· Inside a serif heading, for one word the designer wants to lean into.
· In her spoken memory · quoted text she is recalling to you verbatim.

That is the whole list. No italic placeholder text. No italic captions. No italic page titles. No italic section labels.

Weights used, in total

Serif: 400, 500. That is all.

Sans: 400, 500, 600. 700 only for tabular numerals.

Anything outside this list is not part of the system.

03 · Color in type

When text is rust, and when it is ink.

Ink (#1E1A18) is the voice. Rust (#7B5544) is the accent. Exactly five places rust is allowed on type.

1 · One word in an H1 or H2

A single word or short phrase inside a heading may be rust. Used at most once per page.

Good morning, Arnav.

2 · Section labels (caps)

The uppercase micro-label above a section title is rust · the one consistent categorical use of color. It tells you "this is a new chapter".

3 · The accent button label

The text inside the accent-filled button is paper-colored, not rust. Rust on type appears only as text, never as a button fill swap.

4 · Inline links (rare)

An in-prose link is rust, no underline, with a border-bottom: 1px solid rgba(123,85,68,0.3). Used only where a click is the natural next step · not for navigation.

5 · Live status dot text

When a status chip says "Listening" or "Thinking" · i.e. she is actively present · the chip label is rust with a pulsing dot. Otherwise the chip label is ink-500.

Everywhere else

Ink. Page titles, body, labels, buttons, chat, metadata, captions, nav items. The serif upright is already doing the work of weight · color would be a second signal doing the same job.
